## About

This portfolio features a sleek, responsive design built with React, TypeScript, and Vite. It includes animated 3D elements using React Three Fiber, smooth transitions with Framer Motion, and an intuitive navigation system.

## Features

- **Interactive 3D Background**: Animated particle system with mouse interaction
- **Responsive Design**: Optimized for all screen sizes
- **Modern UI**: Clean, professional design with smooth animations
- **Multiple Sections**:
  - Home: Hero section with quick facts
  - About: Personal story and focus areas
  - Projects: Showcase of key projects including ClipFlow AI, Travel Planner, and more
  - Experience: Internship and certification details
  - Skills: Technical stack and expertise
  - Education: Academic background
  - Contact: Get in touch via email, phone, or social media

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **3D Graphics**: React Three Fiber, Three.js, Drei
- **Animation**: Framer Motion
- **Routing**: React Router
- **Styling**: CSS with modern design patterns
- **Icons**: Lucide React
- **Form Integration**: EmailJS for contact functionality

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/saivighnesh2190/portfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
# Build for production
npm run build

# Preview the production build
npm run preview
```

### Linting

```bash
# Run ESLint
npm run lint
```
